# ESLint Configuration for TCFU

This project uses ESLint with TypeScript for code quality and consistency.

## Available Scripts

- `npm run lint` - Check for linting issues
- `npm run lint:fix` - Fix automatically fixable linting issues
- `npm run typecheck` - Run TypeScript type checking

## Configuration

- **ESLint Config**: `eslint.config.mjs` (flat config format)
- **Rules**: Optimized for Phaser.js game development with TypeScript
- **VSCode Integration**: `.vscode/settings.json` provides automatic fixing on save

## Key Features

- Modern flat config format (ESLint 9+)
- TypeScript-specific rules with game development optimizations
- Import organization and consistency rules
- Phaser.js-friendly rule adjustments
- VSCode integration for seamless development

## Game-Specific Adjustments

- Relaxed `any` usage warnings (common with Phaser APIs)
- Disabled unsafe type checks for Phaser's dynamic properties
- Console warnings instead of errors (useful for debugging)
- Flexible destructuring rules for array-heavy game operations
