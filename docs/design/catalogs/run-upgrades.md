# Catalog: Run upgrades

In-run **W-cell** progression for the current weapon tier ladder. Resets on run end; **kept across life loss**.

## Currency

| Field | Value |
|-------|-------|
| Name | W-cells |
| Scope | Current run only |
| Spend UI | None (auto at thresholds) |

## Earn rates (by enemy class)

| class | w_cells |
|-------|---------|
| fodder | 1 |
| grunt | 2 |
| elite | 5 |
| set_piece | 15 |

Arsenal meta rank 3: **+10%** W-cell earn.

## Tier thresholds (cumulative W-cells this run)

| Reach tier | Cumulative W-cells |
|------------|--------------------|
| 1 | 20 |
| 2 | 50 |
| 3 | 100 |

Max tier **3**. HUD shows tier pips + progress toward next.

## What upgrades

Weapon pattern/stats for the kit’s `weapon_id` only — see [`weapons.md`](./weapons.md) tiers 0–3.
